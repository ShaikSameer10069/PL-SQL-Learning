# 📚 Day 01: SQL Basics & Getting Started

Welcome to **Day 01** of the SQL learning series! Is folder me hum database aur tables banane ke bilkul fundamental concepts cover kar rahe hain. 

## 🎯 Objective
Aaj ka main goal hai samajhna ki database kaise set up hota hai, usme tables kaise design hoti hain, data kaise insert karte hain, aur zaroorat padne par us data ko filter karke kaise nikalte hain.

## 🧠 Concepts Covered
Is module me humne in primary SQL commands ko detail me use kiya hai:

### 1. Data Definition (Structure Banana)
* **`CREATE DATABASE`**: Ek naya database server pe initialize karne ke liye.
* **`CREATE TABLE`**: Database ke andar rows aur columns ka structure banane ke liye. Humne basic data types jaise `INT`, `VARCHAR`, aur `DECIMAL` ka use karke schema design kiya hai.

### 2. Data Manipulation (Data Daalna)
* **`INSERT INTO`**: Banayi gayi table me naya data (records) add karne ke liye. Humne single aur multiple rows insert karna dekha hai.

### 3. Data Querying (Data Padhna)
* **`SELECT`**: Table se data fetch karne ke liye. Isme `SELECT *` (saare columns) aur sirf specific columns fetch karne ke examples hain.
* **`WHERE`**: Data ko specific conditions ke base pe filter karne ke liye (jaise kisi specific department ka data ya ek certain age se upar ka data nikalna).

## 📂 Folder Structure & Files

Is folder me tumhe do main SQL files milengi:

* 📄 **`examples.sql`**: Is file me aaj ke saare concepts ke practical examples step-by-step likhe hain. Ek dummy `Employees` table banayi gayi hai jisme data daal kar alag-alag `SELECT` aur `WHERE` clauses apply kiye gaye hain. Ise tutorial ki tarah use karein.
* 📄 **`practice.sql`**: Yeh file hands-on practice ke liye hai. Isme ek `Students` table ka chhota sa task diya gaya hai jisme table creation se lekar data filtering tak ki queries khud se implement ki gayi hain.

## 🚀 How to Run These Queries?
1. Apne favorite SQL tool (jaise MySQL Workbench, SQL Server Management Studio, ya online compilers jaise DB Fiddle/SQL Fiddle) ko open karein.
2. Sabse pehle `examples.sql` ka code copy karein, usko run karein, aur output ko analyze karein.
3. Uske baad `practice.sql` ko run karein aur queries ko apne hisaab se modify karke test karein.
